WB Printer IoT Integration

Impresión directa de Etiquetas 2x1 y Guías Adjuntas 4x8

Este módulo extiende las capacidades del módulo de Inventario y Código de Barras de Odoo para integrarse fluidamente con la Caja IoT (IoT Box). Permite enrutar las impresiones a impresoras térmicas locales por usuario sin abrir el diálogo del navegador.

Etiquetas 4x2 Térmicas

Plantillas QWeb diseñadas matemáticamente para impresoras térmicas a 203 DPI, asegurando que los códigos de barras y textos encajen perfectamente en 4x2 pulgadas sin bordes blancos.

Interceptor de Adjuntos (4x8)

Convierte el PDF adjunto de las guías de transporte en un 'Reporte Activo' para que Odoo IoT lo reconozca. Evita descargas manuales e imprime directamente en formato 4x8.

Memoria por Usuario: Al integrarse con ir.actions.report, el módulo utiliza la memoria caché del navegador. Odoo preguntará por la impresora solo la primera vez, agilizando el flujo de trabajo en las mesas de empaque.
